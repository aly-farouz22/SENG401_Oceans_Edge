The game explores the sustainability challenges described in United Nations Sustainable Development Goal 14, which
focuses on protecting and sustainably managing marine ecosystems that are essential for climate regulation, biodiversity,
and human livelihoods.

This project is divided into two main parts:
- `server` - Backend service
- `client` - Frontend application

The following steps must be followed to successfully run the game:

## Prerequisites

The following components need to be initially installed:
- Node.js
- npm

## 1. Clone the Repository

```bash
git clone https://github.com/aly-farouz22/SENG401_Oceans_Edge.git
cd SENG401_Oceans_Edge
```
## 2. Setup and Run the Server

```bash
cd server
npm install
npm run dev
```

## Setup and Run Client

```bash
cd client
npm install
npm run dev
```
